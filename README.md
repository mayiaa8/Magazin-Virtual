# Magazin-Virtual
Aplicația are rolul de a gestiona produsele unui magazin virtual. Aceasta permite administrarea stocului de produse, realizarea vânzărilor și calculcarea valorii totale a produselor disponibile.
1. Adăugare produs:
-introducerea numelui produsului
-introducerea prețului
-introducerea cantității disponibile

2. Afișare produse:
-listarea tuturor produselor existente în magazin
-afișarea detaliilor fiecărui produs (nume, preț, cantitate)

3. Căutare produs
-căutare după nume
-afișarea informațiilor despre produsul găsit

4.Vânzare produs
-selectarea produsului dorit
-scăderea cantității din stoc
-afișarea valorii totale a vânzării
